# Portfolio Personal

Este es el repositorio de mi portafolio personal web, disenado para presentar mis proyectos de desarrollo de software, ciencia de datos y automatizacion.

## Tecnologias Utilizadas

El portafolio ha sido construido utilizando tecnologías web estándar y prácticas de Clean Code:

* HTML5 Semantico
* CSS3 Modularizado (Variables, Grid, Animaciones 3D)
* JavaScript puro (Vanilla JS) para interacciones dinamicas

## Caracteristicas Principales

* Interfaz moderna con efecto Glassmorphism y temas oscuros.
* Carrusel 3D interactivo para mostrar proyectos.
* Diseno totalmente responsivo (Mobile First).
* Sistema de gestion de recursos graficos optimizado.
* Arquitectura CSS modular para facil escalabilidad y mantenimiento.

## Estructura del Proyecto

* /css: Modulos de hojas de estilo (variables, layout, components, animations, main).
* /images: Recursos graficos organizados por categorias (projects, skills, social, profile).
* index.html: Documento principal y estructura semantica.
* script.js: Logica interactiva del portafolio.
* cv.pdf: Documento descargable del curriculum.
